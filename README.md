# vuepress-theme-ajwi

## Install

```bash
$ npm install vuepress-theme-ajwi --save-dev
# or
$ yarn add vuepress-theme-ajwi --dev
```

## Usage

```js
// .vuepress/config.js
module.exports = {
  theme: 'ajwi',
  themeConfig: {
    // Please head documentation to see the available options.
  },
}
```

## Development

- **Live Example** - clone this repo and run `npm run example:dev`
